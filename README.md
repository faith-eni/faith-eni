# Hi, I'm Faith 👋

I'm an independent ML researcher and engineer based in London. My current work is on **tiny-object detection in aerial imagery** — measuring how much of the gap between published mAP numbers and reproducible ones comes from data choices vs. architecture. First-author manuscript under review at Springer Nature.

### 📌 Featured project

**[tiny-birds](https://github.com/faith-eni/tiny-birds)** — TIDE-Aerial
A data-centric pipeline for detecting 10–20 pixel objects in 21 MP aerial imagery. Multi-seed evaluation (n=3 seeds per configuration), bootstrap 95% CIs, Wilcoxon paired tests, cross-dataset validated on WBWF, VisDrone, and AI-TOD-v2. Full reproduction stack open-sourced — code, configs, logs, figures.
